# Jegyzet hozzáadás/frissítés

## 1. Pull requesteken keresztül frissíthetőek a jegyzetek

## 2. A commit messagek legyenek egyértelműek:
- tartalmazza, hogy létrehozol (init) vagy frissítesz egy jegyzetet (update)
- tartalmazza a fájl repositoryból nézett elérési útját
- egyéb extra információt

**Például:** `update(1-felev/prog-1/matrixfeladat.md): megoldas pelda`

## 3. Lehetőség szerint markdown fileokat használj
## 4. Próbáld kisebb blokkokba tördelni jegyzeteket
